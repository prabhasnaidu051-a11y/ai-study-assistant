# AI Study Assistant - User Manual

## Features

- Ask AI questions
- Upload PDF documents
- Ask questions from uploaded PDFs
- Generate quizzes

## Running Backend

```bash
cd backend
uvicorn main:app --reload
```

## Running Frontend

```bash
cd frontend
python -m http.server 5500
```

## Open Application

Frontend:
http://localhost:5500

Backend API:
http://127.0.0.1:8000/docs
